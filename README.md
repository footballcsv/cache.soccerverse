## Intro

Football data for France (and Monaco) includes:

| Level |                               |
| ----: | ----------------------------- | 
|     I |  Ligue 1                      |
|    II |  Ligue 2                      |

Note: The French Division 1 and Division 2 changed
to Ligue 1 and Ligue 2 starting with the 2002/03 season.
